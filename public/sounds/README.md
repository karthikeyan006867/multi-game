# Sound Files for Mega Gaming Platform

Place your sound files in this directory:

## Required Sound Files:

- `click.mp3` - Button click sound
- `win.mp3` - Victory sound
- `lose.mp3` - Game over sound
- `coin.mp3` - Collecting coins/points
- `powerup.mp3` - Power-up collection
- `explosion.mp3` - Explosion effect
- `jump.mp3` - Jump action
- `hit.mp3` - Hit/collision sound
- `combo.mp3` - Combo multiplier
- `achievement.mp3` - Achievement unlocked
- `levelup.mp3` - Level up sound
- `background.mp3` - Background music (looping)

## Free Sound Resources:

- **Freesound.org** - https://freesound.org/
- **OpenGameArt** - https://opengameart.org/
- **Zapsplat** - https://www.zapsplat.com/
- **Mixkit** - https://mixkit.co/free-sound-effects/

## Format Recommendations:

- Use MP3 format for compatibility
- Keep file sizes under 500KB
- Sample rate: 44.1 kHz
- Bitrate: 128-192 kbps

## Quick Setup:

If you don't have sound files, the game will work without audio.
For testing, you can use placeholder silence or download free sounds from above resources.

## Adding Custom Sounds:

1. Place your .mp3 files in this directory
2. Update sound names in `lib/sound-manager.ts` if needed
3. Adjust volume levels as needed
